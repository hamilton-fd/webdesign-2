# Hamilton & Forge Digital — Website

**Strategic • Creative • Results-Driven**  
*Apple-inspired, brand-led, performance-focused web presence*

---

## Overview

This repository contains the source code for the **Hamilton & Forge Digital** official website.

The site is designed as a **premium brand experience**, reflecting our philosophy:

> Authority over noise.  
> Precision over volume.  
> Impact over vanity.

It combines **Apple-inspired layout principles** with **Hamilton & Forge’s gold-and-black identity** to deliver a modern, restrained, high-confidence digital presence.

---

## Brand & Design System

The website adheres to the following design principles:

- **Clarity-first layout** inspired by Apple.com
- **Generous spacing and hierarchy**
- **Minimalist typography with strong contrast**
- **Gold accent system** representing authority and legacy
- **Dark, cinematic brand imagery**
- **No UI clutter, no unnecessary motion**

### Colour Palette

- Black / Soft Black — primary foundation  
- Gold — brand authority accent  
- Light Grey — Apple-style neutral canvas  
- Muted Greys — secondary text and structure

## Visual Assets

Brand imagery is stored locally and used intentionally as **hero statements**, not decoration.
